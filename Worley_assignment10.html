Monday's presentation

This presentation was about chapter 13 Fonts and printing. In it, they explained how a page should be formatted and also how to incorporate different styles or even custom fonts into your web page. The group also showed how you can use CSS to setup a web page so it will print out in a easy to read format.

key topics

  • Common styling for printing a page
  • CSS styles to help print a page
  • Embedded fonts/Custom fonts

Wednesday's presentation

This presentation was about chapter 12 Embedding audio and video. The group explained the most common media types for video and audio and went into a brief history for each type of format. The group explained how to make an object for embedding audio/video into a page and some of the attributes required so that the browser is able to understand the type of file. They also showed an example of the new embed tag that is in html5.

key topics

  • Common media files types
  • Codecs for video and audio
  • Embedding with an object
  • Embedding with the embed tag

Friday's presentation

This presentation was about chapter 10 Tables. The group gave detailed information on tables and how they work, explaining how the elements work and some of the CSS styling commands that go with creating tables. They also showed how you can use the pseudo classes in CSS to simplify your table styling. They explained how you have to use html in order to merge columns or rows together and how you add captions to a table.

key topics

  • Table elements and their tags
  • Table formatting with CSS
  • Merging columns and rows with html
  • CSS pseudo commands for styling a table
  • Captions for a table
  • Wrapping control in a table
