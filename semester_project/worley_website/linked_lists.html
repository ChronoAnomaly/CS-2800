Types of Linked Lists

The Linked List is one of the more common and simple types of data structures. It can be designed to work in many different way, some of which allows of different types of accesses and improved efficiency. They are commonly used to store data in a random order as items get added to the list. They can also be combined into other data structures to form hybrid structures that can combine the efficiencies of both types and help improve overall results in different categories. Efficiency

The efficiency for a linked list is fairly similar across different implementations except for some improvements in adding/removing an item to the end of a list. The following table shows the efficiency for operations on a linked list.

Operation Efficiency
Indexing Θ(n)
Insert/delete at beginning Θ(1)
Insert/delete at end Θ(n) when last element is unknown
Θ(1) when last element is known
Insert/delete in middle search time Θ(1)
