Monday's presentation

This presentations was about chapter 14 animations. The group gave detailed information for animations, transitions, transforms, and filters, along with the different types of properties that belong to the different types. Examples were provided that showed how to use the effects from an animation, along with an activity to implement a web page that had various kinds of effects. The key topics are listed in the following list.

Key topics

  • Transitions
  • Transforms
  • Animations
  • Filters

Transitions are a property that can be set to trigger each time an object is changing appearance. They are used to set the duration, a timing function, delay, and what actually changes with an object. The timing function is used tell how fast a transition occurs.

Transforms uses css to change the appearance of an object. The properties related to this function are normally used to change how a picture looks. It is able to rotate, change the scale, skew, or move an image on the page.

Animations are used to change the location of an object. They also allow content to move moved in from outside of a single window, allowing more content to fit into a single page. There are a number of properties related to the time of the animation, direction, its name, and if it repeats.

Filters allow for manipulation of an image after it has already been loaded by the browser. This type has a number of properties used to apply different styles to the image.
